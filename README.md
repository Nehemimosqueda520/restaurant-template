# Restaurant Template
This is a restaurant template built with Next.js and Tailwind CSS. It provides a simple and responsive design for showcasing your restaurant's menu, location, and contact information.
## Features
- Responsive design using Tailwind CSS
- Next.js for server-side rendering and routing
- Customizable components for menu items, location, and contact information
- Dark mode support
## Installation
1. Clone the repository:
```bash
git clone
```
2. Navigate to the project directory:
```bash
cd restaurant-template
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
5. Open your browser and go to `http://localhost:3000` to see the template in action.
## Customization
You can customize the template by editing the components in the `app` directory. The main components include:
- `home`: the landing page of the restaurant.
- `Dishes`: Shows the restaurant's dishes.
- `reserve`: Allows customers to make reservations.
- `Contact`: Provides contact information and a contact form.

You can also modify the styles in `globals.css` to match your restaurant's branding.
